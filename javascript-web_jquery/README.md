# JavaScript - Web jQuery

This project contains scripts demonstrating jQuery for DOM manipulation and AJAX requests.

## Requirements
- Chrome (version 57.0)
- jQuery version 3.x
- semistandard compliant with --global $

## Tasks
0. **0-script.js** - Updates header color using vanilla JavaScript
1. **1-script.js** - Updates header color using jQuery
2. **2-script.js** - Updates header color on click event
3. **3-script.js** - Adds red class on click event
4. **4-script.js** - Toggles classes on click event
5. **5-script.js** - Adds list item on click event
6. **6-script.js** - Updates header text on click event
7. **7-script.js** - Fetches Star Wars character name
8. **8-script.js** - Fetches and lists Star Wars movies
9. **9-script.js** - Fetches hello translation
